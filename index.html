Lumi ‚Äì AI seller call simulator

Lumi is a tool that lets new account managers practice seller calls in a safe environment. It simulates a seller persona, responds in near real time and can evaluate how the call went. I designed the conversation flow, built the front end in HTML, CSS and JavaScript, connected it to n8n for workflow automation and wired in AWS services (speech, LLMs, knowledge bases) to make it actually feel alive.

Conversation design HTML/CSS/JS n8n automation AWS (Polly, Transcribe, LLMs)

Vellaro ‚Äì AI fencing analysis

Vellaro is a machine learning based concept that analyses fencing footage and turns movement into feedback. I defined the product concept for an AI-based fencing feedback tool using pose estimation, conducted interviews with fencers and coaches to identify UX needs and mapped the feature progression from raw pose data to movement metrics and tactical insight.

Pose estimation Sports analytics Product discovery UX research
